# 聯華液位換算表

純前端手機網頁工具，不需要登入、不需要後端、不需要 Google 試算表。

## 功能

- 輸入各桶槽 `1cm=kg`、`目前液位`、`灌後液位`
- 自動計算 `所打 cm` 與 `預估 kg`
- 可輸入 `總淨重`
- 顯示 `已充填` 與 `剩餘`
- 使用瀏覽器 `localStorage` 保存資料
- 離開畫面後再回來，資料仍保留
- `清空` 才會清除資料
- `復原` 可回到上一個輸入狀態

## GitHub Pages

把本資料夾內容放到 GitHub repository 根目錄後，啟用 GitHub Pages：

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/ (root)`

網址會是：

```text
https://<github-username>.github.io/<repository-name>/
```
